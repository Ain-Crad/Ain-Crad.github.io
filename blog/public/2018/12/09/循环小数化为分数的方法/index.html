数学解法

设x = 0.abcabcabc…，则1000x = abc.abcabcabc…。
1000x - x = abc.abcabcabc… - 0.abcabcabc…，即999x = abc。所以$$x = \frac{abc}{999}$$999的因子有1，3，9，27，37，111，333，999，因为a，b，c各不相同且0 < n < 100，所以n的可能取值为27和37。
